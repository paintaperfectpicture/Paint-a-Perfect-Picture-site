# Paint A Perfect Picture LLC — Official Website

This repository contains the official website for Paint A Perfect Picture LLC, a residential, commercial, and industrial painting company based in Columbus, Ohio.

---

## Live Website
https://paintaperfectpicture.com

---

## Project Purpose
This website allows customers to:

• View services offered  
• Contact the company  
• Request quotes  
• Learn about the business  
• View FAQs and service areas  

The site is designed to be fully editable through Netlify CMS without requiring coding knowledge.

---

## Editing the Website (IMPORTANT)

To edit your website content, go to:

https://paintaperfectpicture.com/admin

From there, you can edit and update:

• Phone number  
• Email address  
• Service areas  
• Frequently Asked Questions (FAQs)  
• Services offered  
• Homepage content  

All changes update automatically on your live website after saving.

---

## Project Structure

/  
index.html → Main website homepage  
/admin/ → CMS dashboard  
/content/ → Editable website content  
netlify.toml → Netlify deployment configuration  
README.md → Project documentation  

---

## Deployment

This website is connected to Netlify and deploys automatically.

Any changes pushed to this GitHub repository will instantly update the live website.

No manual deployment required.
